This code is used to invert reciever function waveform to 1D S-wave velocity structure
UnetSe.ipynb : training UNet 
read_UnetSe-Li.ipynb : test the UNet using data of Li et al(2017,tectonophysics)
traindata_4.25.dat: train data
test_data_Li.dat: data of Li et al(2017, tectonophysics
Please find the training data at https://zenodo.org/record/5758500#.Yasr2NBBxPY
